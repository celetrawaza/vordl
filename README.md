# Vordl
Калька игры [Wordle](https://www.nytimes.com/games/wordle/index.html) (Вордли)
с настраиваемыми параметрами и перечнем слов.

# Сборка

## Подготовка набора данных
Определите перечень слов в `data/words_raw.txt` и перечень букв в `data/letters.txt` (на каждую строку по одной букве с соответствующими дубликатами)

## Сборка проекта
```sh
make rebuild
```
Артефакты сборки попадают в папку `out`.

# Запуск
После [сборки](#сборка):
```sh
cd out/
chmod +x vordl
./vordl
```

Параметры запуска:
- `--port число` - порт веб-сервера
- `--wordNumber число` - номер слова, используется как сид для генератора случайных чисел
